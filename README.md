## 📖 이번 프로젝트에서 복습한 것
- 가상선택자를 활용하여 css로 가상 태그를 만드는 법을 공부
- position 속성의 위치를 결정하는 방식을 공부
- float 레이아웃과 clear 속성에 대해 공부
- 절대경로와 상대경로의 차이점과 경로설정 방법을 공부
<br><br>

## ⚙ Refactoring
- reset, common, main, fonts로 나누어 기존 css 파일의 수를 줄임.
- 공백과 줄바꿈을 제거하고 용량을 줄이기 위해 min.css파일로 제작.
